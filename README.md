# NLP_spring26

WP2 demo notebook for moving from Bag of Words to contextual embeddings.

## Environment Setup

Use the PowerShell bootstrap script in the project root to create the `nlp_env` virtual environment and register the notebook kernel:

```powershell
.\setup_nlp_env.ps1 -CreateOnly
```

Student-facing setup and run instructions are in `documents/venv-setup.md`.
